VS PAINT - List Of Features

Group Name Icon Shortcut Definition and Usage
File New Ctrl + N Create new canvas.
Open Ctrl + O Open existed image.
Save Ctrl + S Save the current canvas to the current file.
Save As Ctrl + Shift + S Save the current canvas to a new file.
Export Export the current canvas to .png or .jpeg file.
Import Import a .png of .jpeg file to the current canvas.
Mode Draw B Change into drawing mode.
Erase E Change into erase mode.
Select S Change into selecting mode.
Tool Undo Ctrl + Z Undo latest action.
Redo Ctrl + Y Redo previous action.
Select All Ctrl + A Select every shape.
Deselect All Ctrl + D Deselect every shape.
Transform Ctrl + T Change position/size/rotate selected shape(s).
Copy Ctrl + C Copy selected shape(s).
Paste Ctrl + V Paste selected shape(s).
Layer Move Up Move up selected shape(s).
Move Down Move down selected shape(s).
